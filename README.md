# Lá da Roça - Portal de Solidariedade Comunitária

Este projeto é uma **Extensão Universitária** desenvolvida para a disciplina:  
**COOPERAÇÃO: HUMANISMO SOLIDÁRIO, REDES E COMUNIDADES.**

## 🎯 Finalidade do Projeto
O objetivo principal é utilizar a tecnologia para fortalecer redes de cooperação local. O portal transforma a loja física **Lá da Roça** (Samambaia-DF) num **Hub de Solidariedade**, facilitando a logística de doação e troca de itens essenciais entre os moradores da comunidade.

### Funcionalidades do Portal
* **Vitrine Solidária:** Divulgação de produtos artesanais e caseiros que fomentam a economia local.
* **Ponto de Coleta e Troca:** Seção dedicada a informar sobre a arrecadação de cestas básicas, roupas e itens de higiene sob o conceito "Pegue se precisar, Doe se puder".
* **Mapeamento Local:** Localização exata da unidade matriz e pontos de feira (Vicente Pires e Asa Sul).
* **Comunicação Direta:** Integração com WhatsApp para dúvidas e coordenação de doações.

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido com tecnologias modernas de front-end para garantir performance e responsividade:
* **React + TypeScript**
* **Vite** (Build tool)
* **Tailwind CSS** (Estilização)
* **Lucide React** (Ícones)
* **Framer Motion** (Animações de interface)
* **Shadcn/UI** (Componentes de interface)

## 🚀 Como Rodar o Projeto

1.  **Instalar dependências:**
    No terminal, dentro da pasta do projeto, execute:
    ```bash
    npm i
    ```
   

2.  **Iniciar o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
   

3.  **Aceder ao navegador:**
    O site estará disponível em `http://localhost:5173`.
